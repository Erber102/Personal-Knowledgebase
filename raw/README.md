# raw/ —— 你的原始输入（私有，不进公开仓库）

这个目录放**不可变的源材料**：论文 PDF、网页剪藏、课程讲义 / 习题 / 考试等。

- 系统对 `raw/` **只读、绝不修改**（见 CLAUDE.md 硬规则）。
- `raw/**` 默认被 [.gitignore](../.gitignore) 整体忽略 —— 这些大多是**受版权的他人材料和你的私人笔记**，不应该 commit 到公开仓库。仓库里只保留这个目录骨架。

## 建议的子目录约定

```
raw/
├── papers/                 # 论文 PDF
├── clips/                  # 网页文章剪藏（+ 配图子文件夹）
└── courses/
    └── <COURSE>/           # 每门课一个子目录：讲义 / 章节 PDF / 习题 / 考试
```

把你的材料放进来，然后在仓库根目录用 Claude Code ingest（见根目录 [README.md](../README.md)）。
