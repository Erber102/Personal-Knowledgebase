# examples/ —— 格式示例

这里的文件**只是格式参考**，让你（或第一次看到这个系统的人）直观了解每种页面长什么样。

- [example-paper.md](example-paper.md) —— 一篇 paper 摘要页（基于公开经典论文 DDPM）。对应 `wiki/papers/`。
- [example-concept.md](example-concept.md) —— 一个 concept 概念页（通用概念 score function）。对应 `wiki/concepts/`。

真正使用时，你**不需要手写**这些页 —— 在仓库根目录启动 Claude Code，按 [../README.md](../README.md) 的命令别名 ingest 论文 / 概念，Claude Code 会按 `templates/research/` 下的模板自动生成到 `wiki/` 对应目录。

> 注：这两个示例只演示**格式**，不包含任何原创研究想法。questions / connections / claims 等承载原创 idea 的页面没有放进示例。
